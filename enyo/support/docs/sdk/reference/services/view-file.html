View File

You can view all of the supported file types through the Application Manager service by specifying the target file in the service call in the form of a URI string assigned to the target property. The URI can point to a locally stored file (e.g., file:///filepath.ext), but if it points to a Web-based file (e.g., http://filepath.ext), the file downloads before viewing.

Note:

See Application Manager for the API description and the supported file types.
