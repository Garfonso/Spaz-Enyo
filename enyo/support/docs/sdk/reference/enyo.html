Enyo App Framework APIs

The Enyo API Reference is itself an application built using the Enyo framework.

To view the API Reference in your WebKit-based browser, right-click on the link above and select "Open Link in New Window" or "Open Link in New Tab".

Note: Should you encounter any difficulties in opening the link, use your browser to open the file at <enyo-root>/0.8/support/docs/api/index.html, where <enyo-root> is the directory on your local machine in which you extracted the Enyo archive.
